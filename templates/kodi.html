Method A — Inside Kodi (like The Crew)

Recommended. No Downloader needed.

01
Enable Unknown Sources

Settings → My Fire TV → Developer Options → Apps from Unknown Sources → On

02
Install Kodi

Amazon Appstore → install Kodi.

03
Add Source (File Manager)

Kodi → Settings (gear) → File Manager → Add Source

Type exactly:

nickpelectrical.github.io/percfectai-kodi

Name it PercfectAI → OK

04
Install repo zip from source

Add-ons → box icon → Install from zip file → PercfectAI

Select repository.percfectai-1.0.2.zip

05
Install from repository

Box icon → Install from repository → PercfectAI Repository → Video add-ons → PercfectStudios → Install

06
Connect your TV

Open PercfectStudios → Scan QR with your phone to sign up free (10 images + 6s video).

Method B — Downloader app

Alternative if File Manager method fails.

01
Install Downloader

Amazon Appstore → install Downloader (free, by AFTVnews).

02
Download repo zip

Downloader → URL bar → type exactly:

github.com/nickpelectrical/percfectai-kodi/raw/main/repository.zip

Or short URL: percfectai.com/kodi

03
Install zip in Kodi

Kodi → Add-ons → box icon → Install from zip file

Browse: storage → emulated → 0 → Downloader → repository.zip

04
Install PercfectStudios

Box icon → Install from repository → PercfectAI Repository → Video add-ons → PercfectStudios → Install
