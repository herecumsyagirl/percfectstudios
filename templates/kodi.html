How to Install on Fire Stick

01
Enable Unknown Sources

On your Fire Stick: Settings → My Fire TV → Developer Options → Apps from Unknown Sources → On

02
Install Kodi

Search for "Kodi" in the Amazon Appstore and install it, or sideload via Downloader app.

03
Open Kodi → Add-ons

Launch Kodi, go to Add-ons → click the box icon (top left) → Install from zip file.

04
Enter the download URL

When prompted for a source, type in:

percfectai.com/static/kodi/plugin.video.percfectstudios-1.0.0.zip
05
Enter your API Key

After installing, open the add-on settings and paste your API key below.

06
Start Generating

Open PercfectStudios from your Kodi Add-ons menu and start creating on your TV.
