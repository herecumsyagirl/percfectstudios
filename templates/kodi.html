How to Install on Fire Stick

01
Enable Unknown Sources

On your Fire Stick: Settings → My Fire TV → Developer Options → Apps from Unknown Sources → On

02
Install Kodi

Search for "Kodi" in the Amazon Appstore and install it, or sideload via Downloader app.

03
Open Kodi → Add-ons

Launch Kodi, go to Add-ons → click the box icon (top left).

04
Install from zip file

Type this URL (include https://):

https://percfectai.com/kodi
05
Install from zip folder

If you used the Downloader app instead, go to Install from zip file → Downloads → select the repo zip.

06
PercfectAI Repository installs

Wait for the repository to finish installing. It appears under Install from repository.

07
Install from repository

Add-ons → box icon → Install from repository → PercfectAI Repository → Video add-ons → PercfectStudios → Install.

08
Connect your TV

Open PercfectStudios → Scan QR code with your phone to sign up free. API key below is optional fallback.
